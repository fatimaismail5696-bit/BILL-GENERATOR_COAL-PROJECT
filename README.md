# BILL-GENERATOR_COAL-PROJECT
The 8086 Assembly Bill Generator System is a console-based application developed in 8086 Assembly Language using the EMU8086 Emulator. The project is designed to simulate a simple billing system that performs bill calculations through low-level programming techniques.
The system generates a complete bill by taking item details and performing calculations directly at the assembly level.

## ✨ Project Features

✅ Developed completely in 8086 Assembly Language
✅ Executed and tested on 8086 Emulator
✅ User input handling through interrupts
✅ Item price and quantity calculations
✅ Automatic total bill generation
✅ Console-based interactive interface
✅ Demonstrates low-level programming logic
✅ Uses registers, loops, procedures, and arithmetic instructions

## 🛠️ Technologies & Tools
💻 8086 Assembly Language
⚙️ EMU8086 Emulator
